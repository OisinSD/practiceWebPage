Basic HTML, JavaScript, and CSS webpage.

**Task 1 – Prepare, Design, and Plan**
35 marks
You are required to produce a design document for a website of at least five webpages, which clearly outlines your design approach. The website should have clear and demonstrable interactive features as well as a shopping cart, clear imagery, and at least one animation. 

The design documentation should include wireframes noting the layout of each page, a navigation diagram for the site, and detail any interactive features. The design and website should also have consideration for at least one new HTML5 feature. In addition, your design document should outline three approaches in consideration of search engine optimization.


**Task 2 – Build & Implement Your Design**
50 marks
Using HTML, CSS, and JavaScript, implement your design by building a website with consideration for acceptable coding standards and best practices. The implementation should reflect the requirements and have functional interactive features.


Task 3 – Understanding the Web
15 marks
Write a report as a supplement to your design documentation that:

I. Describes the hardware and software components which enable internet and web services and those chosen for creating your website.
II. Outlines the difference between Web 1.0, 2.0, and 3.0 and the differences in website technologies across the three periods.
III. Details the difference between responsive and non-responsive web design.
